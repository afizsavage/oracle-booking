## Getting Started

- Getting a local copy is simple. Follow the steps below

### Install

- Node
- npm
- git

### Local Copy

- Clone the project

```
git clone https://github.com/afizsavage/oracle-booking
```
## Setup

### For the frontend

```
npm install
```

Start server(frontend):

```
npm start
```

The front end app will open automatically to the default browser

### Run the tests

```
npm run test
```
